Clean, focused **Core Java console programs** built to practice and demonstrate **Object-Oriented Programming (OOP) fundamentals**.

Each project is intentionally named **based on what the code actually demonstrates**, making this repository ideal for:
- Java freshers
- OOP revision
- Interview preparation
- Strong Core Java foundation building
